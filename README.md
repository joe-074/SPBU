# University Projects — Mathematical Robotics & AI (SPbU)

Projects, coursework, and research from the Master’s program **“Mathematical Robotics and Artificial Intelligence” at Saint Petersburg State University**.  
This repository contains implementations, experiments, and assignments in applied mathematics, machine learning, robotics, and intelligent control.

## Contents

This repository includes work covering subjects such as:
- Mathematical Modeling in Cybernetical and Robotic Systems  
- Stochastic Programming  
- Graph Theory  
- High-Performance Computations  
- Mathematical Foundations of Robotics  
- Computer Vision Algorithms  
- Autonomous Navigation and Motion Control  
- Control of Robotic Manipulators  
- Algorithmic Foundations of Robotics  
- Machine Learning  
- Mathematical Models of Signal Processing  
- Research / Technology Projects

## Focus Areas

- Machine Learning and Deep Learning  
- Computer Vision and Robotics Algorithms  
- Optimization and Numerical Methods  
- Motion Planning and Control Theory  
- Simulation and system modeling  
- High-performance scientific computing
